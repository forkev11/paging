## Prerequisites

- Python

## Installation steps

### Clone the repository

- Using SSH
```
git clone git@github.com:forkev11/paging.git
```

- Using HTTPS
```
git clone https://github.com/forkev11/paging.git
```

### Create python venv
```
python3 -m venv ./venv
```

### Activate venv
For sh and bash
```bash
. ./venv/bin/activate
```

For Powershell
```ps1
.\venv\Scripts\Activate.ps1
```

### Install dependencies
```
pip install -r requirements.txt
```

## Run app
```
python app.py
```
